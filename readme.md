# Atreus Firmware

This is my own firmware for the [Atreus keyboard](https://github.com/technomancy/atreus) I designed and built.

## Usage

Edit `layouts.h` using the keycodes defined in `usb_keyboard.h`.

Install `avr-gcc` and run `make upload`.
