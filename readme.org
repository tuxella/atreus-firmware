* Atreus Firmware

This is my own firmware for the [[https://github.com/technomancy/atreus][Atreus keyboard]] I designed and built.

** Usage

Edit =layouts.h= using the keycodes defined in =usb_keyboard.h=.

Install =avr-gcc= and run =make upload=.
